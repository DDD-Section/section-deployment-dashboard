Section Coach — Completion Export

Export per-user course, completion, and certificate data for upload to your LMS (e.g. Workday Learning). Runs entirely in your browser.

Your key is stored only in this browser and is sent only to Section's API (prof.ai) to authorize the data pull. It never goes anywhere else.

Certificates

One row per user × certificate, including completion_date — the only Section field that carries a true completion date, and the grain most LMS completion records use.

Course engagement

One row per user × course: engagement status (Enrolled / Watched Lessons / Completed), session, videos watched, badges earned. Note: course rows carry status, not a completion date.

User summary

One row per user: views, courses completed, badges, plus onboarding and program status.
